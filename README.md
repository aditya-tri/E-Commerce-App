## Tech Stack

**Client:** React, Redux

**Server:** Node, Express

**Database:** MongoDB

## Run Locally

Clone the project

```bash
  git clone https://github.com/aditya-tri/E-Commerce-App.git
```

Go to the project directory

```bash
  cd e-commerce
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
